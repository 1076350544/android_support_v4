android_support_v4
==================

add android feature introduced in newer version.

now support
-----------
* Horizontal ListView
* Horizontal GridView
* Vertical ViewPager
* Center-ViewPager
* Center-ListView
* Center-GridView
* Smooth-scroll in ListView (in response to KeyEvent)

how to build
===================
